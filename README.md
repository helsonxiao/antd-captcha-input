## Installation

```bash
npm install antd-captcha-input
```
or
```bash
yarn add antd-captcha-input
```

## Usage

```tsx | pure
import { CaptchaInput } from 'antd-captcha-input';

const App = () => {
  return (
    <CaptchaInput />
  );
};

export default App;
```

Try it on website: https://helsonxiao.github.io/antd-captcha-input/demos/
